# SEEA Monthly Monitor

Месечно изтегля пълния Excel export от публичния регистър на АУЕР.

Изходни файлове:
- `data/energy_objects.csv`
- `data/energy_objects.xlsx`
- `data/new_records.xlsx`
- `data/last_run.json`

Workflow-ът се изпълнява на 1-во число всеки месец в 06:15 UTC и може да бъде стартиран ръчно от Actions.
